Задание на разработку

Описание технологии

Продукт разрабатывается на Embarcadero Delphi XE7 с использованием СУБД Firebird 2.5.
В текущей версии реализован механизм подключаемых библиотек (далее плагинов), которые дополняют функциональность основных исполняемых модулей без внесения изменений в исходный код.

Тестовое задание необходимо выполнить в среде не ниже Delphi 7  и с применением  компонент входящих в стандартный набор. Допускается   применение дополнительных библиотек с исходным кодом. В качестве примера в задание включен плагин “techsupport”, в модуле проекта которого прописаны все необходимые функции для успешного запуска библиотеки в составе основного приложения.

В комплект задания включены:

  • дистрибутивы Firebird и IBExpert.  
  • Тестовая база данных
  • Исходный код плагина
  • Тестовое приложение проверки работы плагина
  • Описание алгоритма формирования отчета

Задача

Разработать новый плагин отчетности, который на основании выборки текущих остатков из базы данных формирует отчет в формате xls.

Текс запроса содержится в файле «запрос__форма2_№883.txt»

Бланк формы приведен в файле «форма2_№883_.pdf»

Результат

  • Тестовое приложение находит в подкаталоге \Plugins файл DLL и производит его запуск.
  • Интерактивная форма должна позволять выбрать интервал отчета
  • Результат должен быть выведен в файл XLS(XLSX)
  • Должны быть представлены исходный код и скомпилированная библиотека

DB connection credentials

  • login : SYSDBA
  • password : masterkey
  • role : SYSDBA
